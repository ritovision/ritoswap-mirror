# Styling & Branding

The docs site is intentionally branded with the **RitoVision** aesthetic: deep blues, high-contrast accents, and custom typography. The single source of truth today is <code>docs/styles/override.css</code>.

## Nextra overrides

The file also overrides Nextra theme tokens like:

- <code>--nextra-primary-hue</code> and <code>--nextra-primary-saturation</code>
- <code>--nextra-foreground-default</code> for global text color
- <code>--nextra-link-text-color</code> for link states

These are used to align Nextra components (cards, tables, callouts, search, and footer) to the RitoVision palette.

## Practical guidance

- Update global colors in <code>override.css</code> first.
- Keep component-level tweaks small and intentional.
- Use the shared CSS variables for consistency.
