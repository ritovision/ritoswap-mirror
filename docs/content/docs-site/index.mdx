# Docs Site Overview

This workspace documents **all** RitoSwap workspaces, including the docs site itself. It is built with Nextra 4 and branded to match the RitoVision feel.

## Build & deploy

The docs site is deployed on Vercel and includes a Storybook build + sitemap generation in the pipeline.

- <code>pnpm dev</code> runs the docs site locally.
- <code>pnpm build</code> builds the Next.js app.
- <code>pnpm prebuild</code> runs <code>pnpm --filter ritoswap run build-sb:docs</code> to publish Storybook into <code>docs/public/storybook-static</code>.
- <code>pnpm postbuild</code> runs <code>docs/scripts/generate-sitemap.mjs</code> and <code>pagefind</code> for search.
- <code>pnpm test</code> runs Vitest for docs-only tests.

<Callout type="info">
  Vercel deployment is configured in <code>docs/vercel.json</code> and uses <code>pnpm build</code> as the build command.
</Callout>

## Branding source of truth

The docs theme and RitoVision styling are centralized in <code>docs/styles/override.css</code>. Other files exist in <code>docs/styles</code>, but override.css is the canonical source today.
