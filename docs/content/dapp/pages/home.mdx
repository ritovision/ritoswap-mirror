# Home Page

The Home page functions as the central hub for RitoSwap’s dApp and multi-modal agentic chatbot showcase, clearly laying out each feature, including cross-chain token swapping, NFT minting and burning, and token-gated messaging, music and RapBotRito AI, while framing the project as a technical proof-of-concept rather than a commercial product. It establishes expectations around the project’s multidisciplinary scope (blockchain interactions, UI/UX, embedded music, AI infrastructure) and directs users to both the official documentation and the GitHub repository for deeper examination.

## Music

This page features **Rito Rhymes'** track **"Altcoin Love"**, a parody remix of *California Love*. It’s an anthem to the spirit of altcoins and celebrity in the crypto universe, delivered with a fun yet infotaining vibe. While RitoSwap is a deep technical showcase, this music reminds users it’s also a creative project meant to be enjoyed, not just examined.
